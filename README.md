# Repository Search App

Учебный проект — Android-приложение для поиска GitHub-репозиториев с реализацией Clean Architecture и корутин.

## Что реализовано

- Поиск репозиториев с debounce (задержка 500 мс)
- LazyColumn для отображения списка
- Загрузка данных из локального JSON (assets)
- Состояния загрузки (Loading / Ready / Error)
- Индикатор прогресса во время поиска
- Отмена предыдущего запроса при новом вводе

## Технологии

- Kotlin + Coroutines
- Jetpack Compose
- Clean Architecture
- Kotlinx.Serialization (JSON)
- Coil (загрузка изображений)

## Примечание

Проект представлен без Gradle-файлов и настроек сборки — только исходный код для ознакомления с архитектурой и логикой.
